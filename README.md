Instructions to Run Application

## Step 1:  Install node_modules

```bash
# using npm
npm Install

# If npm install gives dependency error due library version conflicts than use.

npm Install --legacy-peer-deps


# OR using Yarn
yarn 
```

## Step 2: Start your Application


### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
